# 知守 KnowGuard · 企业知识库

本地优先的企业资料管理 + AI 问答工具。资料导入、检索、存储全程在本机完成，仅在使用 AI 问答 / 摘要等功能时按需调用大模型接口。

## 下载

前往本仓库上方 Releases 页面下载最新版试用包。当前版本 v0.1.0（试用版），文件为 知守-试用包-0.1.0.zip（约 400MB，含安装包与三份使用文档）。

## 系统要求

Windows 10 / 11（64 位）。无需预装 Python、Node 或数据库，安装包已自带运行时与向量模型。仅 AI 问答 / FAQ / 摘要 / 看图讲解需联网，导入、检索、存储全程可离线。

## 快速开始

解压 zip 后双击 知守-Setup-0.1.0.exe 安装。若出现「Windows 已保护你的电脑 / 未知发行者」提示，点「更多信息，仍要运行」。打开后进入「系统配置」，填入自备的大模型 API Key（默认支持 DeepSeek）并测试连接，然后导入资料、等待自动向量化，即可在「AI 对话」提问。详细步骤见压缩包内《快速上手指南》与《使用说明书》。

## 关于未签名提示

当前试用版尚未做代码签名，首次运行 Windows 可能提示「未知发行者」。这是新软件的正常现象，不影响使用，点「仍要运行」即可，正式版本将补上数字签名。

## 授权

安装后自动开始 7 天全功能试用，无需任何操作。转正式授权时，在「系统配置，授权激活」复制机器码发给我们，我们发回授权串粘入即可解锁（授权绑定本机）。

## 反馈与支持

试用中的问题、建议或需求，欢迎通过本仓库 Issues 或直接联系我们反馈。反馈时附上界面截图、操作步骤与报错文字，可帮助更快定位。
